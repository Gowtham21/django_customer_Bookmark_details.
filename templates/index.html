This is a sample customer bookmark project

In this customer bookmark project I have created three models called

  1. customer_profile
  2. Bookmark
  3. customer_bookmark

In the navbar we have all three model details
In the first two models we have option to do CRUD(Create, Read, Update and Delete) operations
In the third model we have first two tables records along with that search option,
The search option will retrieve records from the table what you want exactly.
The final option we have admin on the right side of navbar feel free to login superuser
Username : Admin
Password : password

In the customer_profile model
we have given four fileds called
  • customer_fname
  • customer_lname
  • lat (Latitude)
  • lon (Longitude)
In the Bookmark model
we have given five fileds called
  • Title
  • URL
  • source_name
  • pub_date
  • customer (ManyToManyField)
In the customer_bookmark model
we have given two fileds called
  • cb_customer_id (ForeignKey)
  • cb_bookmark_id (ForeignKey)
